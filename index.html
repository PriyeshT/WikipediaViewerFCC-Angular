Wikipedia Viewer

This project is part of [Free Code Camp]'s(www.freecodecamp.com) Intermediate Front End Development Projects. I have used AngularJS for this project.

Code Structure

HTML - index.html - This is the home page of the project

CSS - main.css - Stylesheet for the project

SCRIPTS - app.js - Main AngularJS code file.

Use

On the home page, Click on the Search icon to launch a new search or Click on the Surprise Me link to open a Random article from Wikipedia.

If you click on the search icon and enter the search term, you will get the list of top 10 articles that matches your search criteria. Click on any of the list item and it will take you to the Wikipedia page for that item.
